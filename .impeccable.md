## Design Context

### Users
Primary users are end-users finding real-world locations on a map. They need to quickly position the marker, understand the resolved address with confidence, and confirm the location without confusion.

### Brand Personality
Minimal, clear, and trustworthy. The experience should feel precise and calm, while still giving expressive feedback when state changes happen.

### Aesthetic Direction
Light-mode-first, minimal surfaces with one intentional accent layer. UI should remain clean and compact, but key interaction states (pending, resolving, error, ready) should be more prominent and easier to parse at a glance.

### Accessibility & Motion
Target WCAG AA contrast and focus visibility. Use subtle transitions only, and avoid distracting effects.

### Quality Bar
Flagship polish: meticulous spacing, hierarchy, copy consistency, and state clarity.

### Design Principles
1. Prioritize state clarity over decoration.
2. Keep surfaces minimal, and use emphasis sparingly.
3. Make key actions immediately understandable.
4. Ensure readable contrast and visible keyboard focus.
5. Prefer subtle, purposeful motion for feedback.
